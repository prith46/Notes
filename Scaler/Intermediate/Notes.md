## Matrix Rotation

### 90 degree Rotation
1. Transpose the matrix.
2. Reverse each row.

### 180 degree rotation
1. Reverse each row
2. Reverse the entire matrix

     [or]

1. Reverse all rows
2. Reverse all columns

### 270 degree rotation
1. Transponse the matrix
2. Reverse each column
